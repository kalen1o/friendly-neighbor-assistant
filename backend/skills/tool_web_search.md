type: tool
enabled: true
---

## When to use
When the user asks about current events, weather, prices, news, sports scores, stock prices, or anything requiring live/up-to-date data.

## Parameters
- query: The search query. Expand abbreviations and be specific for better results.
- max_results: Number of results (default: 3)

## Instructions
1. Search DuckDuckGo with the optimized query
2. Fetch content from the top result for fresh data
3. Always cite the source URL in your response
