## When to use
When the user asks to calculate something, convert units, or do any math operation.

## Parameters
- expression: The math expression to evaluate

## Instructions
1. Evaluate the mathematical expression safely
2. Show the expression and result clearly
3. For unit conversions, show both the input and output units
