## When to use
When the user asks about the current date, time, what day it is, timezone conversions, or date calculations.

## Parameters
- timezone: Optional timezone name (default: UTC). Examples: "Asia/Ho_Chi_Minh", "US/Eastern", "Europe/London"

## Instructions
1. Return the current date and time in the requested timezone
2. For date calculations, show both the calculation and the result
