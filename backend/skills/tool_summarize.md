name: summarize

type: tool
enabled: true
---

## When to use
When the user asks to summarize content, create a TL;DR, or condense information.

## Parameters
- text: The text to summarize
- style: Summary style — "brief" (1-2 sentences), "detailed" (bullet points), "executive" (key takeaways)

## Instructions
1. Identify the key points and main ideas
2. Produce a concise summary in the requested style
3. Preserve important details like names, numbers, and dates
