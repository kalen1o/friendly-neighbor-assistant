## When to use
When the user asks to translate text to another language, or writes in one language and wants a response in another.

## Instructions
- Identify the source and target languages
- Translate accurately, preserving tone and meaning
- For ambiguous abbreviations: vi = Vietnamese, en = English, fr = French, de = German, ja = Japanese, ko = Korean, zh = Chinese, es = Spanish, ru = Russian, th = Thai
- If the user says "translate to vi", translate to Vietnamese
- Preserve formatting (bullet points, paragraphs)
- Do NOT wrap the translation in code blocks or markdown code fences
- Return the translated text as plain text, not inside ``` blocks
